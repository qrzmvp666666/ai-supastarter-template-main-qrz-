欢迎阅读本项目文档。本文档覆盖从克隆项目、本地开发到生产部署的完整流程。

## 项目结构

本项目是基于 **supastarter** 的 Next.js monorepo，包含以下子应用：

| 应用 | 目录 | 默认端口 | 说明 |
|------|------|----------|------|
| SaaS 应用 | `apps/saas` | 3000 | 主产品（登录、订阅、设置等） |
| 营销站点 | `apps/marketing` | 3001 | 公开页面、博客、定价等 |
| 文档站 | `apps/docs` | 3002 | 本文档站 |
| 邮件预览 | `apps/mail-preview` | 3003 | 开发期间邮件模板预览 |

## 快速导航

- **快速开始** — 安装依赖、启动本地环境
- **认证配置** — OAuth、Magic Link、Passkey
- **支付配置** — Stripe、Lemon Squeezy 等 5 个提供商
- **数据库迁移** — Prisma 迁移命令
- **对象存储** — MinIO 本地 / S3 生产配置
- **生产部署** — Vercel、Railway 等平台部署指南
